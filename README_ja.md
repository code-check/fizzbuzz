### フィズバズ問題

Write a function to implement FizzBuzz Logic.
フィズバズロジックを実行する機能を書いてください。
フィズバズロジック,
もしある数字が
-3で割り切れ、"Fizz"で置き換えられる。
-5で割り切れ、"Buzz"で置き換えられる。
- 3と5で割り切れるものは、"FizzBuzz"で置き換えられる。
-3でも5でも割り切れない、もしくは15でも割り切れないは何も変わらない。

### 例

<table>
  <tr>
    <td><b>Input</b></td>
    <td>1</td>
    <td>3</td>
    <td>5</td>
    <td>15</td>
  </tr>
  <tr>
    <td><b>Output</b></td>
    <td>1</td>
    <td>Fizz</td>
    <td>Buzz</td>
    <td>FizzBuzz</td>
  </tr>
</table>
