### FizzBuzz Challenge

1から100までの数を表示させる。  
ただし3の倍数の時は数字の代わりにFizz、5の倍数の時は数字の代わりにBuzz、3と5両方の倍数の時はFizz Buzzと表示させるプログラムを書きなさい。

### 例

<table>
  <tr>
    <td><b>インプット</b></td>
    <td>1</td>
    <td>3</td>
    <td>5</td>
    <td>15</td>
  </tr>
  <tr>
    <td><b>アウトプット</b></td>
    <td>1</td>
    <td>Fizz</td>
    <td>Buzz</td>
    <td>FizzBuzz</td>
  </tr>
</table>
